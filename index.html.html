ABOUT

QA is the last line of defense before the user finds the bug.

I'm a Software Test Engineer based in Mumbai, with four-plus years testing web and mobile products end to end — from reading the BRD to signing off the release. I've worked across food delivery, e‑commerce, travel booking, and hospital management systems, which means I've tested almost every kind of flow a user actually cares about: login and OTP, checkout and payment, booking and cancellation, scheduling and billing.

My day-to-day is grounded in the fundamentals — functional, regression, integration, smoke and sanity testing, written and executed by hand — plus enough API testing in Postman to chase a bug into the backend when the UI alone won't tell the full story. I work inside Agile teams, I log clean defects in Jira, and I care about shaving down the time between "found" and "fixed."
